
admin - містить файли панелі адміністрування магазинів, то, що бачить адміністратор сайту
	controller - папка для OpenCart контролерів, містить контролери для роботи з каталогом (товар, категорія, атрибути, опції, фільтри ітд.), основні (common) контролери (шапка сайту, футер, меню, сторінка авторизації, ліва колонка ітд.), контролери розширень (модулі, модифікації, оплата, доставка ..), локалізація, встановлений модулі, настройки, робота з користувачами ітд.
	language - папка мовних файлів, містить папки для кожної мови з мовними файлами для цієї мови
	model - папка для моделей
	view - папка для уявлень, шаблонів, містить як самі шаблони так і картинки для них, яваскрипт бібліотеки і файли, css файли
	config.php
	index.php
	php.ini
catalog - містить файли каталогу магазину, то, що бачить користувач, коли заходить на сайту
	controller
	language
	model
	view папка для уявлень, шаблонів, містить як самі шаблони так і картинки для них, яваскрипт бібліотеки і файли, css файлиможет містити тільки один шаблон
image - містять задану зображення для товарів, каталогів, прапори, зображення для товара без зображення, кешування копії зображень.
log для Логування помтлок
system - містить системні файли
	cache - папка для кешування даних
	config - папка для конфігураційних файлів
	download - папка для завантажених файлів
	engine - файли для основних класів движка: дія action, контролер controller, події event, фронт контролер front, завантажувач loader, модель model, реєстр registry
	helper - файли функцій помічників: json - для перетворення масивів в json об'єкти, utf8 - для роботи з текстом в кодуванні utf8, vat - для vat кодів
	library - класи бібліотеки: драйвера баз даних db, системи кешування cache, робота з кошиком cart, файлами config, користувачами user, валютою currency, зображеннями image, поштою mail, користувачами user, кодування encryption ітд.
	logs - логи помилок

	modification - файли модифікацій
	upload - файли для вивантаження
	modification.xml - файл модифікацій OpenCart
	startup.php - файл для перевірки установок системи і завантаження основних класів і помічників, необхідних для OpenCart. Цей файл вантажиться одним з перших при завантаженні системи.
 
.htaccess.txt - містить налаштування для ЧПУ (людино зрозумілі URL) (щоб посилання на сайті були yoursite.com/index.php?route=product/product&product_id=40, a yoursite.com/iphone), а також деякі настройки для сервера, наприклад час виконання скриптів, максимальний розмір для завантаження ітд.
config.php - конфігураційний файл, містить визначення глобальних змінних для папок сайту, наприклад DIR_APPLICATION для папки каталогу, DIR_SYSTEM для папки системних фалів ітд., а також налаштування бази даних: драйвер БД, хост, користувач, пароль, БД, префікс таблиць
crossdomain.xml - файл междоменной політики, потрібен наприклад для Adobe флеш плеєра.
index.php - головний файл, який запускається першим
php.ini - файл php налаштувань, тут можна задати налаштування наприклад для стандартної кодування, ліміту оперативної пам'яті для сайту ітд.
